Bu site 2017 yılında GitHub üzerinden host edilmek üzere Jekyll sistemi ile oluşturuldu.

BLOG ana sayfası (index.html), BİLGİSAYAR BİLİMLERİ ana sayfası (csmain.html) ve HAKKIMDA/İLETİŞİM (about.html) sayfaları HTML5 ile yazılıp CSS3 ile şekillendirildi.

Bunların dışında, bu üç ana sayfadan bağlantıları verilen içerik sayfaları, markdown sayfa yapısı kullanılarak oluşturuldu. Tümü, browser'daki linkin sonuna .md uzantısı eklenerek indirilebilir.

Site, BLOG sayfasında fikir ve sanat eserlerinin, BİLGİSAYAR BİLİMLERİ sayfasında yazarın GitHub üzerinden paylaşıma sunduğu kodların içerilmesi düşünülen bir kişisel site / blog olarak planlandı.

Bu tarihe kadar, caglayandemirci.net gibi bir domain ismine gerek duyulmadı, sitenin bir GitHub platformu olduğunu vurgulayan default isim kullanılmaya devam ediyor.
