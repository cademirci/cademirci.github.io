Bu site 2017 yılında GitHub üzerinden host edilmek üzere Jekyll sistemi ile oluşturuldu.

BLOG ana sayfası (index.html), BİLGİSAYAR BİLİMLERİ ana sayfası (csmain.html) ve HAKKIMDA/İLETİŞİM (about.html) sayfaları, üzerinde bulunduğunuz bu sayfa ve bazı istisnai sayfalar HTML ile yazılıp CSS ile şekillendirildi.

Bunların dışında, bu üç ana sayfadan bağlantıları verilen içerik sayfaları, Markdown sayfa dili kullanılarak oluşturulup CSS ile şekillendirildi.

Site, BLOG sayfasında fikir ve sanat eserlerinin, BİLGİSAYAR BİLİMLERİ sayfasında yazarın GitHub üzerinden paylaşıma sunduğu kodların içerilmesi düşünülen bir kişisel site / blog olarak planlandı.

"Hayat, Evren ve Her Şey'e dair Nihai Kişisel Blog" şeklindeki slogan, Douglas Adams'ın yazdığı Otostopçunun Galaksi Rehberi (The Hitchhiker's Guide to the Galaxy) adlı kitapta, güçlü bir bilgisayara buldurulmaya çalışılan "hayat, evren ve her şeye dair nihai cevap (42)" kültüne göndermede bulunuyor.

Arka plan resmi: In Flames, Sounds of a Playground Fading albüm kapağındaki karga ve saat figürü.

Bu tarihe kadar, caglayandemirci.net gibi bir domain ismine gerek duyulmadı, sitenin bir GitHub platformu olduğunu vurgulayan default isim kullanılmaya devam ediyor.

***

GitHub üzerinde bulunduğundan, site herhangi bir server manipülasyon programı (PHP gibi) içermiyor. Git kullanımının sağladığı üzere, tüm sayfalar ve dosyalar kişisel bilgisayar üzerinde muhafaza edilebilir, yönetilebilir ve paylaşıma sunulmak istendiğinde birkaç komut yardımıyla site sahibinin GitHub Repository'sine eklenebilir. Linux ya da MacOS kullanılmadığı zaman bile, bu işlemler Windows üzerinde Git Bash kullanılarak yapılabilir.

Aynı şekilde, Jekyll ve GitHub ile alakalı olarak, Markdown sayfa dili de büyük kolaylıklar sağlayan; Word editörüne yazı yazar gibi web sayfaları oluşturabilmeye yarayan bir yapı olduğundan, site bir HTML manipülasyon programı da içermiyor. Yalnız, Markdown sayfalarını uygun isimle uygun klasöre ekleyen bir python programı var.
